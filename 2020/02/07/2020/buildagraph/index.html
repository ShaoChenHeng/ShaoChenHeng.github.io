图的建立以及使用数据结构存储

2020-02-07 graphtheory algorithmanddatastruct

邻接表存图

邻接表存图实际上是用链表来存储边的相关信息,适用于稀疏图.

struct edge
{
int to, next, w;
} s[1000];
int n, num, head[1000];

void build( int u, int v, int w )
{
s[++num].w = w;
s[num].next = head[u];
s[num].to = v;
head[u] = num;
}

void work()
{
int u, v, w;
for ( int i = 0; i < n; i ++ )
{
scanf(“%d %d %d”, &u, &v, &w);
build( u, v, w );
}
}

首先定义一个edge的结构体,以及head[]

  • to为当前边指向的下一个结点.
  • w为当前边的权重.
  • next为在同一层的边的下一条边的编号,同一层的边也就是连接当前结点和它的子节点(们)的(那些)边.
  • head为一个表头,记录邻接表的每一”行”首个元素边的编号,或者说同一层的那些边的首个边的编号.

每添加一条新的边就记录下它所指向的下一条边和这条边的权重

s[num].next = head[u]; head[u] = num;

这两句代码是将当前边插入同一层所在的那个链表,并且插入后的数据与输入的数据是倒序的.

二维矩阵存图

二维矩阵适合存稠密图,但是能够存储的结点会相对少一些,实现方法非常简单.

int G[100][100]

void work()
{
int from, to, weight;
for ( int i = 0; i < E; i ++ )
{
scanf(“%d %d %d”,&from, &to, &weight);
G[from][to] = weight;
}

}

STL向量存图

实现方法也很简单,如果想要记录权重等信息则需要建立新的结构体.

void work()
{
vector G[1000];
int u, v;
for ( int i = 0; i < E; i ++ )
{
scanf(“%d %d”, &u, &v);
G[u].push_back(v);
}
}
